\section{\textbf{Question}}
(Geometry,1.10) Q. Using cosine formula in an equilateral triangle, show that $\cos60^\circ=\frac{1}{2}$.
\section{\textbf{Explanation}}
Let us consider an equilateral $\triangle{ABC}$, and consider midpoint of line segment AC as F. A line segment is drawn from B to F which bisects Line AC. The figure is shown as below:
\renewcommand{\thefigure}{1}
\begin{figure}[!ht]
    \centering
    \resizebox{7cm}{!}{\input{TriangleFig.tex}}
    \caption{Equilateral $\triangle{ABC}$}
    \label{fig:tri_right_angle}
\end{figure}

We know that in equilateral triangle all sides have equal length.
\begin{align}
    \label{eq:1}
    \implies\norm{AB}=\norm{BC}=\norm{AC}
\end{align}

Since F is midpoint of line AC.
\begin{align}
    \implies\norm{AF}=\frac{1}{2}\norm{AC}
\end{align}
From \eqref{eq:1},
\begin{align}
    \label{eq:2}
    \implies\norm{AF}=\frac{1}{2}\norm{AB}
\end{align}

$\triangle{ABF}$ forms a right-angled triangle. Using cosine formula,
\begin{align}
    \implies\cos60^\circ=\frac{\norm{AF}}{\norm{AB}}\\
    =\frac{\frac{1}{2}\norm{AB}}{\norm{AB}}\\
    \implies\cos60^\circ=\frac{1}{2}
\end{align}
Hence proved.
